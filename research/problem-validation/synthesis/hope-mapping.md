# Pain → HOPE 기능 매핑

| Pain Point | 근거 (키워드) | HOPE 기능 | Gain |
|------------|---------------|-----------|------|
| 가정 피드백 부재 | 부모, 가정, 피드백 | 몬스터 대결 발음 정확도·음소 피드백 | 즉각 피드백 |
| 반복 연습 동기 부족 | 연습, 아동, 게임 | 피치/짝맞추기 게임, 보상·출석 | 재미·지속 |
| 비용·접근 부담 | 비용, 대기, 기관 | 앱 내 가정학습 (병원 외 연습) | 접근성 |
| 일반화 어려움 | 일반화, 보호자 | 다양한 게임·단어 라운드 | 상황 전이 |
| 보호자 참여 | 보호자, 리포트 | 보호자 설정·주간 리포트 | 참여·가시성 |
| 데이터 부재 | 분석, AI | Speech Coach API 연동 | 객관적 점수 |

## HOPE가 해결하는 문제 (한 문장)

> 치료실 밖에서도 **재미있게 연습**하고 **즉각 피드백**을 받을 수 있는 가정 기반 언어재활 보조 서비스

## 공모전 스토리라인

1. **사회문제** — 텍스트마이닝·공공데이터로 Pain 정의
2. **근거** — 뉴스/논문/통계 교차 검증
3. **솔루션** — HOPE 앱 데모
4. **검증** — 파일럿·인터뷰 (추후)

분석 수치는 `outputs/` 산출물을 PPT에 삽입하세요.
