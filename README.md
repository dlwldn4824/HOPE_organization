# HOPE_organization

TEAM 정융고능팀 레츠고

아이의 발음을 함께 듣는 시간 — HOPE speech-learning web app.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS v4
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run lint` — run ESLint
